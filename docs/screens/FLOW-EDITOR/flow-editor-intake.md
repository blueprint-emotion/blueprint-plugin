# FLOW-EDITOR Intake

## 서비스
기획자용 플로우 시각화 플랫폼. MD 파일을 업로드하면 react-flow로 플로우를 그려주고, 편집·저장·히스토리 관리를 지원한다.

## 사용자
IT 서비스 기획자 (개발 경험 없을 수 있음)

## 화면 목적
플로우를 업로드·편집·저장·히스토리 관리하는 단일 화면

## 레이아웃
- 좌측 사이드바: 노드 목록 (그룹별 정렬, 드래그&드롭으로 그룹 간 이동), 하단에 플로우 업로드 버튼
- 우측 상단 툴바: 히스토리 셀렉트박스, 가져오기 버튼, 저장 버튼, 되돌리기 버튼, 정렬 버튼
- 우측 메인: react-flow 캔버스
- 모달 2개: 플로우 업로드 (형식 검증), 저장 버전 선택

## 핵심 행동
- MD 파일 업로드 → 형식 검증 → 캔버스에 미리보기 (DB 저장 아님)
- 노드 수정/삭제, 엣지 레이블 수정/삭제
- 좌측 노드 목록에서 드래그로 그룹 간 노드 이동
- 저장 시 3가지 선택: 엎어쓰기 / 메이저 버전 올리기 / 마이너 버전 올리기
- 최초 저장은 메이저 기본, 1.0부터 시작
- 히스토리 셀렉트박스에서 과거 버전 선택 후 가져오기
- Cmd+Z로 한 단계 되돌리기
- 최초 상태로 되돌리기 버튼
- 오토 레이아웃 정렬 (react-flow 기능)

## 제약사항
- 업로드 시 정해진 MD 형식에 맞는지 검증 필요
- 업로드는 캔버스 미리보기까지만, 저장은 별도 액션
- viewport: pc
