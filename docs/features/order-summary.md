# 주문 요약

## 와이어프레임 요소

| 요소 | type | 설명 |
|------|------|------|
| 선택 상품 수 | text | 선택된 상품 개수 표시 |
| 상품 금액 | text | 선택된 상품들의 합계 금액 |
| 배송비 | text | 배송비 표시 (조건부 무료 배송 안내 포함) |
| 할인 금액 | text | 적용된 할인 총액 (있을 경우) |
| 총 주문 금액 | text | 최종 결제 예정 금액 (굵게 강조) |
| 주문하기 버튼 | button | 주문서 작성 페이지로 이동 (Primary CTA) |

## 상태

| 상태 | 설명 |
|------|------|
| 기본 | 선택된 상품 기준으로 금액 계산되어 표시 |
| 선택 없음 | 금액 모두 0원, 주문하기 버튼 비활성화 |
| 무료 배송 | 배송비 "무료" 표시 + 무료 배송 기준 안내 |

## 인터랙션

- 장바구니 상품 선택/수량 변경 시 → 실시간 금액 재계산
- 주문하기 버튼 클릭 → 선택된 상품으로 주문서 작성 페이지 이동
- 선택된 상품이 없을 때 주문하기 클릭 → "상품을 선택해주세요" 안내

## 비즈니스 로직

- 선택된 상품만 금액에 포함 (체크 해제된 상품은 제외)
- 배송비 정책: 일정 금액 이상 무료, 미만 시 배송비 부과
- 품절 상품은 금액 계산에서 자동 제외
