# 주문 요약

## 와이어프레임 요소

| 요소 | type | 설명 |
|------|------|------|
| 선택 상품 수 | text | 선택된 상품 개수 표시 |
| 상품 금액 | text | 선택된 상품의 합계 금액 |
| 할인 금액 | text | 적용된 할인 총액 |
| 배송비 | text | 배송비 표시 (무료배송 조건 충족 시 "무료") |
| 총 결제금액 | text | 최종 결제 예정 금액 (강조 표시) |
| 결제하기 버튼 | button | 결제 페이지로 이동 |

## 상태

| 상태 | 설명 |
|------|------|
| 기본 | 선택된 상품 기준으로 금액 표시 |
| 선택 없음 | 모든 금액 0원, 결제하기 버튼 비활성화 |

## 인터랙션

- 결제하기 버튼 클릭 → 결제 페이지로 이동
- 상품 선택/수량 변경 시 → 금액 실시간 재계산

## 비즈니스 로직

- 무료배송 기준: 50,000원 이상
- 기본 배송비: 3,000원
- 선택된 상품이 0개이면 결제 불가
