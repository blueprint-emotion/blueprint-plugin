# 상품 검색

## 와이어프레임 요소

상품 목록 화면 상단에 배치되는 검색 요소 목록.

| 요소 | type | 설명 |
|------|------|------|
| 검색 입력 | input | 상품명 또는 키워드를 입력하는 검색 필드 |
| 검색 버튼 | button | 입력한 키워드로 검색을 실행하는 버튼 |
| 결과 요약 문구 | text | 현재 검색 결과 개수 또는 검색어를 보여주는 안내 문구 |

## 상태

| 상태 | 설명 |
|------|------|
| 기본 | 검색어 없이 전체 상품 수를 안내 |
| 검색 적용 | 검색어와 결과 수를 함께 표시 |
| 결과 없음 | 검색 결과가 없음을 알리는 문구 표시 |

## 인터랙션

- 검색 입력 후 검색 버튼 클릭 → 해당 키워드로 상품 목록 갱신
- 검색어가 없으면 전체 상품 목록 기준으로 결과 요약 표시

## 비즈니스 로직

- 검색은 상품명과 핵심 키워드를 기준으로 수행한다
- 검색 결과 수는 현재 필터 조건과 함께 반영된다
