# 버전 관리

## 저장

### 와이어프레임 요소

| 요소 | type | 설명 |
|------|------|------|
| 모달 제목 | text | "버전 저장" 제목 텍스트 |
| 현재 버전 덮어쓰기 | radio | 현재 버전에 덮어쓰는 옵션 (예: 1.1 → 1.1) |
| 메이저 버전 올리기 | radio | 메이저 번호를 증가시키는 옵션 (예: 1.1 → 2.0) |
| 마이너 버전 올리기 | radio | 마이너 번호를 증가시키는 옵션 (예: 1.1 → 1.2) |
| 버전 미리보기 | text | 선택한 옵션에 따른 저장될 버전 번호 표시 |
| 저장 버튼 | button | 선택한 옵션으로 버전 저장 실행 |
| 취소 버튼 | button | 모달 닫기, 저장하지 않음 |

### 상태

| 상태 | 설명 |
|------|------|
| 첫 저장 | 메이저 버전 올리기 선택됨 (기본값), 버전 미리보기 "1.0" 표시. 현재 버전 덮어쓰기·마이너 버전 올리기 비활성 |
| 기본 | 세 가지 옵션 모두 활성, 현재 버전 덮어쓰기 선택됨 (기본값) |
| 저장 중 | 저장 버튼 비활성 + 로딩 표시, 옵션 변경 불가 |
| 저장 완료 | 모달 닫힘, 변경된 버전 번호 반영 |

### 인터랙션

- 옵션 선택 변경 → 버전 미리보기에 저장될 버전 번호 즉시 반영
- 저장 버튼 클릭 → 선택된 옵션으로 버전 저장 → 모달 닫힘
- 취소 버튼 클릭 → 모달 닫힘, 변경 없음
- 모달 외부 클릭 → 모달 닫힘, 변경 없음

### 비즈니스 로직

- 첫 저장 시 메이저 버전 올리기만 선택 가능, 버전 1.0으로 시작
- 첫 저장 이후에는 세 가지 옵션 모두 선택 가능
- 현재 버전 덮어쓰기: 버전 번호 변경 없이 내용만 갱신
- 메이저 버전 올리기: 메이저 +1, 마이너 0으로 초기화 (예: 1.3 → 2.0)
- 마이너 버전 올리기: 마이너 +1 (예: 1.1 → 1.2)

## 버전 번호

### 와이어프레임 요소

| 요소 | type | 설명 |
|------|------|------|
| 버전 표시 | text | 현재 버전 번호 표시 (예: "v1.2") |

### 상태

| 상태 | 설명 |
|------|------|
| 미저장 | 버전 번호 없음, "미저장" 또는 빈 상태 표시 |
| 저장됨 | 현재 버전 번호 표시 (예: "v1.0") |

### 비즈니스 로직

- 버전 형식: `메이저.마이너` (예: 1.0, 1.1, 2.0)
- 메이저와 마이너는 0 이상의 정수
- 최초 저장 시 1.0에서 시작
- 버전 번호는 단조 증가, 감소 불가
