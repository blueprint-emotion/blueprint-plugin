# 상품 필터

## 와이어프레임 요소

상품 목록 화면에서 목록 범위를 좁히는 필터 요소 목록.

| 요소 | type | 설명 |
|------|------|------|
| 카테고리 필터 | checkbox | 카테고리별 상품만 보이도록 선택하는 체크박스 그룹 |
| 가격 범위 필터 | select | 가격 구간을 선택하는 드롭다운 |
| 무료배송 필터 | checkbox | 무료배송 상품만 표시하는 체크박스 |
| 초기화 버튼 | button | 모든 필터 조건을 해제하는 버튼 |

## 상태

| 상태 | 설명 |
|------|------|
| 기본 | 필터 미적용 상태 |
| 일부 적용 | 하나 이상의 필터가 선택된 상태 |
| 전체 초기화 | 모든 필터가 해제되고 기본 목록으로 복귀 |

## 인터랙션

- 필터 선택 또는 해제 → 상품 목록을 즉시 갱신
- 초기화 버튼 클릭 → 적용된 필터를 모두 해제

## 비즈니스 로직

- 여러 카테고리 필터를 동시에 선택할 수 있다
- 가격 범위는 한 번에 하나만 선택한다
- 필터 조건은 검색어와 함께 조합되어 적용된다
