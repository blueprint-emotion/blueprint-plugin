# 버전 히스토리

## 버전 목록

### 와이어프레임 요소

| 요소 | type | 설명 |
|------|------|------|
| 버전 선택 | select | 저장된 버전 목록을 보여주는 셀렉트 박스 (툴바 내 배치) |

### 상태

| 상태 | 설명 |
|------|------|
| 기본 | 현재 버전이 선택된 상태, 셀렉트 박스 접힘 |
| 펼침 | 셀렉트 박스가 열려 저장된 버전 목록 표시 |
| 버전 선택됨 | 목록에서 다른 버전을 선택한 상태, 가져오기 버튼 노출 |
| 빈 목록 | 저장된 버전이 없는 상태, 안내 텍스트 표시 |

### 인터랙션

- 셀렉트 박스 클릭 → 저장된 버전 목록 펼침
- 목록에서 버전 선택 → 선택된 버전 표시 + 가져오기 버튼 노출

## 버전 가져오기

### 와이어프레임 요소

| 요소 | type | 설명 |
|------|------|------|
| 가져오기 버튼 | button | 선택된 버전을 현재 편집 영역에 불러오는 액션 버튼 |

### 상태

| 상태 | 설명 |
|------|------|
| 비활성 | 버전이 선택되지 않은 상태, 버튼 숨김 또는 비활성 |
| 활성 | 버전 목록에서 버전을 선택한 상태, 버튼 활성 |
| 로딩 | 선택한 버전을 불러오는 중, 버튼 비활성 + 로딩 표시 |

### 인터랙션

- 가져오기 버튼 클릭 → 선택된 버전의 데이터를 현재 편집 영역에 로드
- 로드 완료 → 편집 영역이 선택한 버전의 상태로 갱신

## 실행 취소

### 와이어프레임 요소

| 요소 | type | 설명 |
|------|------|------|
| 실행 취소 안내 | text | 단축키 안내 텍스트 (Cmd+Z) |

### 상태

| 상태 | 설명 |
|------|------|
| 실행 취소 가능 | 편집 히스토리에 되돌릴 단계가 있는 상태 |
| 실행 취소 불가 | 더 이상 되돌릴 단계가 없는 상태 |

### 인터랙션

- Cmd+Z 입력 → 편집 히스토리에서 한 단계 되돌림
- 되돌릴 단계가 없으면 아무 동작 없음

## 초기화

### 와이어프레임 요소

| 요소 | type | 설명 |
|------|------|------|
| 초기화 버튼 | button | 모든 변경사항을 되돌리고 초기 상태(또는 마지막 로드 상태)로 복원하는 액션 버튼 |

### 상태

| 상태 | 설명 |
|------|------|
| 활성 | 편집 변경사항이 존재하는 상태, 버튼 활성 |
| 비활성 | 변경사항이 없는 상태(이미 초기 상태), 버튼 비활성 |

### 인터랙션

- 초기화 버튼 클릭 → 확인 다이얼로그 표시 ("모든 변경사항이 사라집니다. 초기화하시겠습니까?")
- 확인 → 편집 영역을 초기 상태(또는 마지막 로드한 버전)로 복원
- 취소 → 다이얼로그 닫힘, 현재 상태 유지
