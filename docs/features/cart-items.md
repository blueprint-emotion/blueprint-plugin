# 장바구니 상품

## 와이어프레임 요소

| 요소 | type | 설명 |
|------|------|------|
| 전체 선택 | checkbox | 모든 상품 일괄 선택/해제 |
| 상품 선택 | checkbox | 개별 상품 선택/해제 |
| 상품 이미지 | image | 상품 썸네일 |
| 상품명 | text | 상품 이름 |
| 옵션 정보 | text | 색상, 사이즈 등 선택한 옵션 표시 |
| 수량 조절 | input | −/+ 버튼과 수량 표시 |
| 상품 가격 | text | 해당 상품의 소계 금액 |
| 삭제 버튼 | button | 개별 상품 장바구니에서 제거 |
| 선택 삭제 버튼 | button | 선택한 상품 일괄 삭제 |

## 상태

| 상태 | 설명 |
|------|------|
| 기본 | 담은 상품이 목록으로 표시됨 |
| 빈 상태 | "장바구니가 비어있습니다" 메시지 + 쇼핑 계속하기 링크 |
| 로딩 | 스켈레톤 UI (상품 카드 3개 분량) |
| 품절 상품 | 품절 표시 + 수량 변경 비활성화 |

## 인터랙션

- 전체 선택 체크박스 클릭 → 모든 상품 선택/해제, 주문 요약 금액 재계산
- 개별 상품 체크박스 클릭 → 해당 상품 선택/해제, 주문 요약 금액 재계산
- 수량 + 클릭 → 수량 1 증가, 가격 재계산
- 수량 − 클릭 → 수량 1 감소 (최소 1), 가격 재계산
- 삭제 버튼 클릭 → "삭제하시겠습니까?" 확인 → 확인 시 상품 제거
- 선택 삭제 클릭 → 선택된 상품 일괄 삭제
- 상품명 클릭 → 상품 상세 페이지로 이동

## 비즈니스 로직

- 최소 수량 1, 최대 수량은 재고 수량까지
- 품절 상품은 수량 변경 불가, 주문 금액에서 제외
- 장바구니 최대 100개 상품
