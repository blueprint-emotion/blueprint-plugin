# 장바구니 상품 관리

## 와이어프레임 요소

| 요소 | type | 설명 |
|------|------|------|
| 전체 선택 체크박스 | checkbox | 모든 상품 일괄 선택/해제 |
| 상품 체크박스 | checkbox | 개별 상품 선택/해제 |
| 상품 이미지 | image | 상품 썸네일 이미지 |
| 상품명 | text | 상품 이름 (클릭 시 상품 상세로 이동) |
| 옵션 정보 | text | 선택한 옵션 (색상, 사이즈 등) 표시 |
| 상품 가격 | text | 개별 상품 단가 |
| 수량 조절 | input | -/+ 버튼이 있는 수량 입력 필드 |
| 소계 | text | 상품 단가 × 수량 |
| 삭제 버튼 | button | 개별 상품 장바구니에서 제거 |
| 선택 삭제 버튼 | button | 체크된 상품 일괄 삭제 |

## 상태

| 상태 | 설명 |
|------|------|
| 기본 | 장바구니에 담긴 상품이 목록으로 표시됨 |
| 빈 상태 | "장바구니가 비어있습니다" 메시지 + 쇼핑 계속하기 링크 |
| 로딩 | 스켈레톤 UI (상품 카드 3개 분량) |
| 에러 | "장바구니를 불러오지 못했습니다" + 재시도 버튼 |

## 인터랙션

- 전체 선택 체크박스 클릭 → 모든 상품 선택/해제, 주문 요약 금액 재계산
- 개별 체크박스 클릭 → 해당 상품 선택/해제, 모두 선택 시 전체 선택 체크박스도 체크
- 수량 + 버튼 클릭 → 수량 1 증가, 소계 및 주문 요약 재계산
- 수량 - 버튼 클릭 → 수량 1 감소 (최소 1), 소계 및 주문 요약 재계산
- 삭제 버튼 클릭 → "삭제하시겠습니까?" 확인 → 해당 상품 제거
- 선택 삭제 버튼 클릭 → 체크된 상품 일괄 삭제 확인 → 제거
- 상품명 클릭 → 상품 상세 페이지로 이동

## 비즈니스 로직

- 로그인한 사용자만 장바구니 접근 가능 (비로그인 시 로그인 페이지로 리다이렉트)
- 수량 최소 1개, 최대 99개
- 재고 초과 수량 입력 시 재고 수량으로 자동 조정 + 안내 메시지
- 판매 종료/품절 상품은 목록에 표시하되 선택 불가 + "품절" 배지
