# 플로우

## 플로우 업로드

### 와이어프레임 요소

| 요소 | type | 설명 |
|------|------|------|
| 업로드 버튼 | button | MD 파일 업로드 모달을 여는 버튼 (좌측 사이드바 하단) |
| 파일 선택 | input | MD 파일을 선택하는 파일 입력 필드 |
| 형식 검증 결과 | text | 업로드한 파일의 형식 검증 성공/실패 메시지 |
| 미리보기 확인 버튼 | button | 검증 통과 후 캔버스에 플로우를 렌더링하는 버튼 |

### 상태

| 상태 | 설명 |
|------|------|
| 기본 | 모달 닫힘, 업로드 버튼만 표시 |
| 모달 열림 | 파일 선택 필드 표시, 미리보기 확인 버튼 비활성 |
| 검증 성공 | 성공 메시지 표시, 미리보기 확인 버튼 활성 |
| 검증 실패 | 실패 사유 메시지 표시, 미리보기 확인 버튼 비활성 |

### 인터랙션

- 업로드 버튼 클릭 → 업로드 모달 열림
- 파일 선택 → 자동으로 형식 검증 실행
- 검증 성공 시 미리보기 확인 버튼 클릭 → 모달 닫힘 + 캔버스에 플로우 렌더링
- 검증 실패 시 실패 사유 표시, 다른 파일 재선택 가능

### 비즈니스 로직

- 업로드 파일은 MD 형식만 허용
- 정해진 MD 형식(헤딩, 리스트 구조)에 맞는지 검증
- 업로드 결과는 캔버스 미리보기까지만 반영, DB에 저장하지 않음

## 노드 관리

### 와이어프레임 요소

| 요소 | type | 설명 |
|------|------|------|
| 노드 목록 | list | 좌측 사이드바의 그룹별 노드 목록 |
| 노드 항목 | text | 개별 노드 이름 표시 (드래그 가능) |
| 노드 수정 | input | 캔버스에서 노드 이름을 인라인 수정하는 입력 필드 |
| 노드 삭제 버튼 | button | 선택한 노드를 삭제하는 버튼 |

### 상태

| 상태 | 설명 |
|------|------|
| 기본 | 노드 목록에 그룹별 노드 표시, 캔버스에 노드 렌더링 |
| 노드 선택 | 선택한 노드 하이라이트, 삭제 버튼 표시 |
| 노드 수정 중 | 인라인 입력 필드 활성, 수정 내용 실시간 반영 |
| 드래그 중 | 노드 항목이 드래그 상태, 이동 가능한 그룹 하이라이트 |

### 인터랙션

- 캔버스 노드 클릭 → 노드 선택 상태
- 노드 더블클릭 → 인라인 수정 모드 진입
- 수정 완료(Enter 또는 포커스 이탈) → 노드 이름 업데이트
- 삭제 버튼 클릭 → 노드 삭제 (연결된 엣지도 함께 삭제)
- 좌측 사이드바에서 노드 항목 드래그 → 다른 그룹에 드롭 → 그룹 간 이동

## 엣지 관리

### 와이어프레임 요소

| 요소 | type | 설명 |
|------|------|------|
| 엣지 레이블 | text | 엣지 위에 표시되는 레이블 텍스트 |
| 엣지 레이블 수정 | input | 엣지 레이블을 인라인 수정하는 입력 필드 |
| 엣지 삭제 버튼 | button | 선택한 엣지를 삭제하는 버튼 |

### 상태

| 상태 | 설명 |
|------|------|
| 기본 | 엣지에 레이블 표시, 선택 없음 |
| 엣지 선택 | 선택한 엣지 하이라이트, 삭제 버튼 표시 |
| 레이블 수정 중 | 인라인 입력 필드 활성, 수정 내용 실시간 반영 |

### 인터랙션

- 엣지 클릭 → 엣지 선택 상태
- 엣지 레이블 더블클릭 → 인라인 수정 모드 진입
- 수정 완료(Enter 또는 포커스 이탈) → 레이블 업데이트
- 삭제 버튼 클릭 → 엣지 삭제

## 레이아웃 정렬

### 와이어프레임 요소

| 요소 | type | 설명 |
|------|------|------|
| 정렬 버튼 | button | 캔버스의 모든 노드와 엣지를 자동 정렬하는 버튼 (우측 상단 툴바) |

### 인터랙션

- 정렬 버튼 클릭 → react-flow 오토 레이아웃 알고리즘 실행 → 캔버스 노드/엣지 재배치
