```markdown
# 댓글

## 와이어프레임 요소

| 요소 | type | 설명 |
|------|------|------|
| 댓글 목록 | list | 작성자 아바타, 이름, 시간, 내용 표시 |
| 댓글 입력 | input | 텍스트 입력 영역 (placeholder: "댓글을 입력하세요") |
| 등록 버튼 | button | 댓글 등록 |
| 답글 버튼 | button | 각 댓글 하단에 답글 달기 버튼 |
| 삭제 버튼 | button | 본인 댓글에만 표시되는 삭제 버튼 |
| 멘션 태그 | text | 댓글 내 @사용자명 하이라이트 표시 |

## 상태

| 상태 | 설명 |
|------|------|
| 기본 | 댓글 목록이 시간순으로 표시, 입력창 비어있음 |
| 빈 상태 | "아직 댓글이 없습니다. 첫 댓글을 남겨보세요!" 메시지 |
| 로딩 | 스켈레톤 UI (댓글 3개 분량) |
| 에러 | "댓글을 불러오지 못했습니다" + 재시도 버튼 |
| 등록 중 | 등록 버튼 비활성화 + 스피너 |

## 인터랙션

- 등록 버튼 클릭 → 댓글 목록 상단에 새 댓글 추가, 입력창 초기화
- 답글 버튼 클릭 → 해당 댓글 바로 아래에 답글 입력창 펼침
- 삭제 버튼 클릭 → "정말 삭제하시겠습니까?" 확인 다이얼로그 → 확인 시 삭제
- @ 입력 시 → 멘션 자동완성 드롭다운 표시 (팀 멤버 목록)
- 댓글 목록 스크롤 → 무한 스크롤 (20개씩 로드)

## 비즈니스 로직

- 로그인한 사용자만 댓글 작성 가능 (비로그인 시 입력창 숨김)
- 본인 댓글만 삭제 가능 (타인 댓글에는 삭제 버튼 미표시)
- 댓글 최대 1,000자, 초과 시 입력 차단 + 글자수 카운터 빨간색
- 멘션된 사용자에게 알림 발송
- 답글은 1단계만 허용 (답글의 답글 없음)
```
