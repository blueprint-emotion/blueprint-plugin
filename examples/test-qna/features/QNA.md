# 상품 Q&A

## Q&A 목록

### 비즈니스 로직

- 문의 유형: 상품문의, 배송문의, 교환/반품, 기타
- 답변 상태: 미답변, 답변완료
- 미답변 문의는 접수 후 24시간 이내 답변 원칙
- 비공개 문의는 작성자와 관리자만 열람 가능
- 삭제된 문의는 목록에서 숨김 처리하되 DB에서 제거하지 않음

## 답변 관리

### 비즈니스 로직

- 답변은 1회만 등록 가능, 등록 후 수정 가능
- 답변 등록 시 문의 상태가 자동으로 "답변완료"로 변경
- 답변 삭제 시 문의 상태가 "미답변"으로 복원
