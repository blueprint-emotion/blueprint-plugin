# 상품 그리드

## 와이어프레임 요소

상품 목록 화면 본문에 그려질 카드형 목록 요소.

| 요소 | type | 설명 |
|------|------|------|
| 상품 카드 목록 | list | 여러 상품 카드를 그리드 형태로 나열한 목록 |
| 상품 이미지 | image | 각 상품의 대표 이미지 영역 |
| 상품명 | text | 각 상품의 이름 |
| 가격 정보 | text | 할인가 또는 판매가를 보여주는 텍스트 |
| 상품 태그 | text | 무료배송, 베스트 등 요약 태그 |
| 장바구니 버튼 | button | 해당 상품을 장바구니에 담는 버튼 |

## 상태

| 상태 | 설명 |
|------|------|
| 기본 | 여러 상품 카드가 그리드로 표시 |
| 결과 없음 | 조건에 맞는 상품이 없다는 빈 상태 메시지 표시 |
| 로딩 | 카드 스켈레톤 또는 플레이스홀더 표시 |

## 인터랙션

- 상품 카드 클릭 → 상품 상세 화면으로 이동
- 장바구니 버튼 클릭 → 해당 상품을 장바구니에 추가

## 비즈니스 로직

- 상품 카드는 현재 검색어 및 필터 조건에 따라 달라진다
- 품절 상품은 장바구니 버튼 대신 상태 문구로 대체될 수 있다
