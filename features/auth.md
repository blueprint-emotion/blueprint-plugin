# 인증

## 와이어프레임 요소

회원가입 화면에 그릴 UI 요소 목록.

| 요소 | type | 설명 |
|------|------|------|
| 서비스 소개 문구 | text | 회원가입 화면 상단의 안내 문구 |
| 이름 입력 | input | 사용자의 이름 또는 표시 이름 입력 필드 |
| 이메일 입력 | input | 가입에 사용할 이메일 주소 입력 필드 |
| 비밀번호 입력 | input | 비밀번호 설정 입력 필드 |
| 비밀번호 확인 입력 | input | 비밀번호 재입력 확인 필드 |
| 약관 동의 체크박스 | checkbox | 서비스 이용약관 및 개인정보 처리방침 동의 |
| 회원가입 버튼 | button | 입력한 정보로 계정을 생성하는 주요 액션 버튼 |
| 소셜 회원가입 버튼 | button | 외부 계정으로 간편 회원가입하는 버튼 그룹 |
| 로그인 이동 링크 | link | 기존 계정 보유 사용자를 로그인 화면으로 이동시키는 링크 |

## 상태

| 상태 | 설명 |
|------|------|
| 기본 | 입력값이 비어 있고 회원가입 버튼은 비활성 또는 기본 상태 |
| 입력 중 | 필수 항목이 채워지고 버튼 활성 조건이 가까워지는 상태 |
| 유효성 오류 | 이메일 형식 오류, 비밀번호 불일치, 약관 미동의 등의 안내 표시 |
| 제출 중 | 회원가입 버튼 비활성화, 진행 상태 안내 표시 |
| 완료 | 가입 성공 후 환영 메시지 또는 다음 단계 안내 표시 |

## 인터랙션

- 회원가입 버튼 클릭 → 필수 입력값과 약관 동의 여부를 검증한 뒤 가입 절차 시작
- 비밀번호 확인 입력 완료 → 비밀번호 일치 여부를 즉시 안내
- 약관 동의 체크박스 선택 → 회원가입 가능 조건에 반영
- 소셜 회원가입 버튼 클릭 → 선택한 외부 계정 가입 절차로 이동
- 로그인 이동 링크 클릭 → 로그인 화면으로 이동

## 비즈니스 로직

- 필수 항목은 이름, 이메일, 비밀번호, 비밀번호 확인, 약관 동의다
- 이메일은 중복 가입이 불가능하다
- 비밀번호는 최소 길이와 문자 조합 정책을 만족해야 한다
- 비밀번호 확인 값은 비밀번호와 정확히 일치해야 한다
- 약관 미동의 시 회원가입을 진행할 수 없다
