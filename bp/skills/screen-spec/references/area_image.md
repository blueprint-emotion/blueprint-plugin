# 이미지 갤러리

## 유저스토리

- [Must] 구매자로서, 이 영역에서 상품의 실제 모습을 여러 각도에서 확인하고 싶다 — 구매 전 실물 감각을 파악하기 위해
- [Should] 구매자로서, 디테일을 확대해서 살펴보고 싶다 — 색상·소재·마감 품질을 짐작하기 위해

**elements**
- PRODUCT__IMAGE 이미지를 갤러리 형태로 표시
  - 대표 이미지: PC 호버 시 확대, 모바일 스와이프
  - 목록 이미지: PC 하단 썸네일, 모바일 인디케이터 도트
- 이미지 카운터 ("N/전체" 형식)
- 이미지 로드 실패 시 플레이스홀더 표시

## 인터랙션

| 구성요소 | 인터랙션 | 상세 |
|---|---|---|
| 대표 이미지 (PC) | 호버 | 마우스 위치 기준으로 확대 영역 표시 (300%) |
| 대표 이미지 (모바일) | 스와이프 | 좌우 드래그로 이전/다음 이미지 전환 |
| 썸네일 (PC) | 클릭 | 클릭한 이미지가 메인 영역으로 전환 |
| 인디케이터 도트 (모바일) | 탭 | 탭한 위치의 이미지로 즉시 이동 |

## 예외 케이스

| 상황 | 처리 방식 |
|---|---|
| 이미지 0장 | "이미지 준비중" 플레이스홀더 + 카운터 숨김 |
| 일부 이미지 로드 실패 | 해당 위치만 플레이스홀더, 나머지는 정상 표시 |

## 인수조건

- [ ] PC에서 메인 이미지 호버 시 확대 영역이 표시된다
- [ ] 모바일에서 좌우 스와이프로 이미지를 탐색할 수 있다
- [ ] 썸네일 클릭 시 메인 이미지가 전환된다
- [ ] 이미지 로드 실패 시 플레이스홀더가 표시된다
- [ ] 키보드 좌우 화살표로 이미지 전환이 가능하다
